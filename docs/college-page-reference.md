# مرجع تصميم صفحات الكلية

هذا المرجع يعتمد صفحة:
`/dashboard/college/subjects`
كنموذج رسمي لبناء الصفحات الأخرى داخل مسار الكلية.

## الهدف

توحيد الشكل المؤسسي للصفحات بحيث تكون كل الصفحات الجديدة بنفس:
- الهوية البصرية
- توزيع المحتوى
- تجربة الاستخدام

## الهيكل المعتمد (Layout Blueprint)

### 1) رأس الصفحة (Header Card)
- بطاقة مستقلة أعلى الصفحة.
- خصائص التصميم:
  - حواف: `rounded-[22px]`
  - حدود خفيفة: `border-[#E8EEF7]`
  - ظل ناعم: `shadow-[0_8px_24px_rgba(15,23,42,0.05)]`
  - شريط علوي رفيع متدرج أزرق (ارتفاع `3px`)
- المحتوى:
  - عنوان الصفحة (قوي وواضح)
  - وصف مختصر هادئ

### 2) بطاقة البيانات الرئيسية (Main Data Card)
- بطاقة منفصلة تحت الهيدر.
- تحتوي بالترتيب:
  1. شريط الإجراءات (Action Bar)
  2. شريط الإحصاءات
  3. الجدول
  4. Pagination

## شريط الإجراءات (Action Bar)

مكوّنات ثابتة مقترحة:
- زر رئيسي (إضافة)
- زر ثانوي (تصدير)
- حقل بحث
- فلتر نوعي

نمط الأزرار:
- رئيسي أزرق: `bg-[#1E3A8A]` مع `hover:bg-[#172554]`
- ثانوي بإطار: `border-[#CBD5E1] bg-white`

## شريط الإحصاءات

أربع بطاقات صغيرة داخل شبكة:
- إجمالي الكيانات
- إجمالي الفروع/الأنواع الفرعية
- النشط
- آخر إضافة

نمط البطاقة:
- `rounded-2xl`
- `border-[#E5ECF6]`
- خلفية فاتحة: `bg-[#F8FAFC]`

## الجدول الاحترافي

متطلبات ثابتة:
- `thead` بلون خلفية واضح: `bg-[#F1F5F9]`
- صفوف body بخطوط فاصلة ناعمة
- حالة فارغة برسالة واضحة
- عمود إجراءات (3 نقاط) مع قائمة تعديل/حذف
- شارة نوع (Badge) عندما يوجد نوع سجل

## Pagination

يظهر دائمًا أسفل الجدول:
- السابق / التالي
- أرقام الصفحات
- سطر يوضح نطاق النتائج المعروضة من الإجمالي

## قواعد التناسق للصفحات القادمة

1. أي صفحة جديدة في `dashboard/college` يجب أن تتبع نفس الهيكل الثنائي:
   - Header Card
   - Main Data Card
2. الحفاظ على نفس تدرج الأزرق والحدود الخفيفة.
3. اعتماد نفس أنماط الأزرار والمدخلات.
4. منع القفز البصري: المسافات العمودية تكون متقاربة (`space-y-6` غالبًا).
5. أي جدول إداري جديد يجب أن يدعم:
   - بحث
   - فلتر
   - إجراءات صف
   - Pagination

## ملاحظة تنفيذية

عند بناء صفحة كلية جديدة:
1. انسخ هيكل `SubjectsPanel`.
2. بدّل أسماء الحقول والمنطق فقط.
3. لا تغيّر هوية التصميم الأساسية إلا بطلب صريح.
